# Design mockup

The initial desktop-shell mockup is available at [desktop-shell-mockup.svg](desktop-shell-mockup.svg). It establishes the native-window look and required first-screen information architecture before visual implementation begins.

The dashboard is intentionally the launch screen. It foregrounds explicit text status for Twitch, the local overlay, and the task system; task management; the Browser Source URL; and session controls. The left rail maps directly to Dashboard, Tasks, Connections, Appearance, and Help.
